Cómo instalar Windows (10 / 11)

Sigue esta guía paso a paso para instalar Windows 10 u 11 en tu equipo. Asegúrate de contar con una licencia válida y realizar un respaldo de tus datos antes de formatear.

  1. Crear un USB booteable: Descarga la herramienta oficial de Microsoft (Media Creation Tool) o utiliza Rufus para grabar la ISO en un USB.
  2. Configurar la BIOS/UEFI: Ajusta el orden de arranque para que el equipo inicie desde el USB.
  3. Iniciar la instalación: Reinicia tu PC y, cuando aparezca el instalador, selecciona el idioma y la edición de Windows.
  4. Seleccionar disco y particiones: Elige la partición adecuada para instalar Windows. Puedes formatear o crear nuevas particiones.
  5. Proceso de instalación: Espera a que el instalador copie los archivos. El equipo se reiniciará varias veces.
  6. Configuración inicial: Configura tu cuenta, contraseña y opciones de privacidad tras la instalación.
  7. Instalar controladores y actualizaciones: Utiliza Windows Update y descarga drivers desde el sitio del fabricante.
